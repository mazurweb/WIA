Pop up Modal Plugin on Exit

By Michael Mazur

Hey Guys

For my codesentation I decided to cover a good pop up modal plugin that displays when the user goes to close or leave the page!

Cons

So cons for this plugin if not created properly can lead to many different things.

  1. When it pops up it will keep on popping up.
  2. Sometimes it may not popup if not configured properly.
  3. Can be really annoying for visitors if it is shown everytime they refresh or go to a new page.
  4. And so on...

Pros

There are many pros to this plugin and is a great marketing strategy for companies looking to try and save visitors from leaving.

  1. As said before great tool for keeping visitors to stay.
  2. Can get the point across for a marketing strategy when the customer gets ready to leave
  3. Really good to track customer information if used as a form or a portal to another page with a form.

Lets get started!

To get Started

You will want to create a div element with the display of none and the id can be whatever you would like but will be used as a selector for your JQuery

Once you have this together you will throw it right before the close of your body tag!

Create a JS file to store code for the JQuery portion of the code.

For mine I called mine init.js which will call from the SimpleModal plugin that is available for us.

The code below is to add custom functionality

                        
                            $(document).ready(function() {

                            $(document).mouseover(function(e) {
                            if(e.clientY <= 25)
                            {
                            $('#exit_content').modal({onOpen: modalOpen, onClose: simplemodal_close});
                            }
                            });
                            });
                            var once = false;

                            function modalOpen (dialog) {
                            while ( once === false){
                            dialog.overlay.fadeIn('fast', function () {
                            dialog.container.fadeIn('fast', function () {
                            dialog.data.hide().slideDown('fast');
                            });
                            });
                            once = true;
                            }
                            }

                            function simplemodal_close (dialog) {
                            dialog.data.fadeOut('fast', function () {
                            dialog.container.hide('fast', function () {
                            dialog.overlay.slideUp('fast', function () {
                            $.modal.close();
                            });
                            });
                            });
                            }
